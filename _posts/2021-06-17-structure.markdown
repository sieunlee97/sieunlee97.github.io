# 순서 리스트 자료구조

순서 리스트(Ordered List) 또는 선형 리스트(Linear List)는 리스트 기반의 자료구조이다. 배열 자료구조와 비슷한 구조를 가진다.
리스트를 사용하는 대표적인 형태가 순서 리스트인데, 나열된 자료 사이 순서를 가지는 리스트를 말한다. 주로 인덱스 관리에 사용된다.


# 배열 자료구조

배열은 프로그램 언어의 기능 중에서 가장 많이 사용되며, 동일한 형태의 자료를 연속해서 저장하는 구조를 가진다.
